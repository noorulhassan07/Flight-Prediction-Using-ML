STEPS TO RUN THIS PROJECT
------------------------------------------------------------------------------------------------------------------------------------------------
1. Clone the repo:

git clone https://github.com/your-username/flight-delay-prediction.git
cd flight-delay-prediction

------------------------------------------------

2. Install dependencies:

pip install -r requirements.txt

------------------------------------------------

3. Train the model:

py train_model.py

------------------------------------------------

4. Generate charts:

py visualizations/graphs.py

------------------------------------------------

5. Run the web app:

py app.py

------------------------------------------------

6. Visit in browser:

http://127.0.0.1:5000/

------------------------------------------------------------------------------------------------------------------------------------------------
